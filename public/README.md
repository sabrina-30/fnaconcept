# Image Assets for FNA Concept Website

## Required Images

### 1. Hero Background (hero-bg.jpg)
- **Dimensions**: 1920x1080 (Full HD)
- **Format**: JPEG, optimized for web
- **Content**: High-quality image of a renovated interior space
- **File size**: < 300KB (optimized)

### 2. About Section Image (about-image.jpg)
- **Dimensions**: 800x600
- **Format**: JPEG, optimized for web
- **Content**: FNA Concept team or completed renovation project
- **File size**: < 150KB (optimized)

## Optimization Guidelines

1. **Compression**: Use tools like TinyJPG or ImageOptim to compress images
2. **Responsive variants**: Create multiple sizes for responsive images (small, medium, large)
3. **Format**: Use WebP format with JPEG fallback for better compression
4. **Lazy loading**: All images except hero should use lazy loading (already implemented)
